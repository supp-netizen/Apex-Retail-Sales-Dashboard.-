# Apex Retail Sales Dashboard (Excel)

## 📊 Project Overview

The **Apex Retail Sales Dashboard** is an interactive Excel dashboard designed to analyze retail sales performance across different regions, product categories, payment methods, and sales personnel.

The dashboard provides key business insights through **KPIs, slicers, and visual charts**, enabling users to quickly monitor sales trends, profitability, and operational performance.

---

## 🎯 Objectives

* Track overall sales performance.
* Monitor profit generation across product categories.
* Analyze sales distribution by region.
* Compare payment method usage.
* Identify top-performing salespersons.
* Visualize sales trends over time.

---

## 🚀 Dashboard Features

### 1. Interactive Slicers

Users can dynamically filter dashboard data using:

#### Region

* East
* North
* South
* West

#### Payment Mode

* Card
* Cash
* Online

#### Category

* Clothing
* Electronics
* Furniture
* Home Decor
* Other
* Sports

These slicers provide real-time filtering across all dashboard visuals.

---

### 2. Key Performance Indicators (KPIs)

| KPI                 | Description                              |
| ------------------- | ---------------------------------------- |
| Total Sales         | Displays overall sales revenue generated |
| Total Profit        | Shows cumulative profit earned           |
| Total Quantity Sold | Total number of units sold               |

These KPIs update automatically based on slicer selections.

---

### 3. Visualizations

#### Sales by Region

* Clustered Column Chart
* Compares sales performance across regions.
* Helps identify the highest and lowest performing markets.

#### Profit by Category

* Pie Chart
* Displays contribution of each product category to total profit.
* Useful for profitability analysis.

#### Quantity by Payment Mode

* Horizontal Bar Chart
* Shows quantity sold through different payment methods.
* Highlights customer payment preferences.

#### Top 3 Salesperson

* Doughnut Chart
* Displays the top-performing sales representatives based on sales contribution.

#### Sales Trend Over Time

* Line Chart
* Tracks sales performance over a period.
* Helps identify trends, seasonality, and anomalies.

# How to Use

1. Open the Excel workbook.
2. Navigate to the Dashboard sheet.
3. Use the slicers to filter data by Region, Payment Mode, or Category.
4. Review KPI cards for summary metrics.
5. Analyze charts for detailed insights.
